# Hi, I'm Chenul 👋

I'm an Astrophysics student at Carleton University who loves making sense out of chaos! Finding patterns in things that seem unpredictable or random. That's what drew me to data science. I use domain knowledge from astrophysics as a lens for rigorous analytical thinking, and I build projects that span statistical analysis, machine learning, and interactive visualization.

Currently building an AI-powered stock and crypto research agent.

Open to data science co-ops and internships in Ottawa and beyond.

## 🚀 Projects

| Project | Description | Stack |
|--------|-------------|-------|
| [StellarStats](https://github.com/Chenul-Gomes/StellarStats) | Statistical analysis of 120,000 stars — hypothesis testing, regression, clustering, and an interactive HR diagram | Python, scipy, scikit-learn, Plotly |
| [SpikeML](https://github.com/Chenul-Gomes/SpikeML) | Valorant match outcome predictor with data leakage detection, XGBoost, SHAP explainability, and a Streamlit dashboard | Python, XGBoost, SHAP, Streamlit |
| [AnalyticsIQ](https://github.com/Chenul-Gomes/AnalyticsIQ) | E-commerce SQL analytics pipeline with RFM segmentation, cohort retention, and revenue trend modelling | SQL, Python, Plotly |
| [F1-Race-Analysis](https://github.com/Chenul-Gomes/F1-Race-Analysis) | F1 driver analytics pipeline with lap time prediction (MAE: 0.907s, R²: 0.645) | Python, FastF1, scikit-learn |
| [Satellite-Orbit-Simulator](https://github.com/Chenul-Gomes/Satellite-Orbit-Simulator) | Physics-based orbital simulator using Keplerian elements with real-time 3D visualization | Python, PyOpenGL, NumPy |
